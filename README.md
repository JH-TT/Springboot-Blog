## Spring Boot로 블로그 개발하기

---

스프링 부트 3으로 배포와 CI/CD까지 해보고 싶어서 만든 토이 프로젝트 입니다.

## 사용툴

---

- **Spring boot 3.0.2**
  - 기존에는 2버전을 사용했었고 3버전은 어떤지 경험해 보고 싶어서 정했습니다.
- **Correto 17**
- **JUnit5**
  - 단위 테스트도 해보기 위해서 사용했습니다.
- **H2**
  - 로컬에서 진행할때 사용했습니다.
- **MySQL**
  - 서비스로 올릴 때 사용했습니다.
- **스프링 데이터 JPA**
  - Repository 역할을 하는 인터페이스를 만들어줘 CRUD를 간단하게 할 수 있어서 사용했습니다.
- **Thymeleaf**
